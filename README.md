# Four-Capability｜职业教育课程“四能重构”方案生成器

本项目用于将工程技术类课程重构为“四能培养”课程方案，帮助教师从课程内容、岗位任务和实训条件出发，自动生成课程能力培养设计。

## 一、四能框架

| 能力 | 核心含义 | 典型输出 |
|---|---|---|
| 结构翻译能力 | 把真实设备、工艺对象、任务场景或故障现象翻译成专业结构、变量参数、功能关系、图表、符号或模型 | 结构标注图、变量表、任务对象分析表 |
| 运动建模能力 | 根据结构、任务和约束，建立可用于分析、计算、仿真、编程或控制的模型 | 机构模型、控制模型、工艺模型、信号模型、仿真模型 |
| 过程求解能力 | 按“已知条件—目标结果—方法选择—分步实施—结果验证”的流程完成任务 | 计算书、程序文件、调试记录、仿真报告 |
| 工程判断能力 | 判断方案、程序、模型或结果是否满足安全、精度、效率、稳定性、限位、碰撞、成本和工程可执行性要求 | 可行性判断表、风险分析表、评价量规 |

总逻辑：

```text
真实任务输入 → 结构翻译 → 模型建立 → 过程求解 → 工程判断 → 能力输出
```

## 二、适用课程

适用于高职、应用型本科工程技术类课程，例如：

- 机器人技术基础
- 工业机器人编程与调试
- 机械设计基础
- 液压与气动技术
- PLC控制技术
- 数字孪生与虚拟调试
- 机电设备安装与调试
- 传感器与检测技术
- 工业视觉检测
- 智能制造系统

## 四、快速开始

### 1. 命令行生成AI提示词

```bash
python cli.py examples/robotics_basic.json --mode prompt --output output_prompt.md
```

### 2. 命令行生成课程方案骨架

```bash
python cli.py examples/robotics_basic.json --mode scaffold --output output_plan.md
```

### 3. 启动网页工具

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 六、输出内容

本工具可生成：

1. 课程重构定位；
2. 课程内容—四能映射表；
3. 项目化教学重构方案；
4. 每个项目的四能培养任务；
5. 学习成果证据；
6. 评价量规；
7. 教学实施建议；
8. 可直接交给大模型继续扩写的高质量提示词。
